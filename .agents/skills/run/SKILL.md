# Run Portfolio Generator

When the user triggers this skill, you must execute the CLI application to generate and print the cryptocurrency portfolio.
Command: `.\venv\Scripts\python.exe main.py run` (or `python main.py run` if the virtual environment is already activated).

Wait for the process to complete and output the results to the user.
